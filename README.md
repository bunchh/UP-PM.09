# «Блог»

## Инструкция по развертыванию проекта

Для запуска проекта необходимо выполнить следующие действия.

### 1. Клонирование репозитория

Сначала необходимо скачать проект с GitHub:

git clone https://github.com/bunchh/UP-PM.09

После этого перейти в папку проекта.

### 2. Настройка веб-сервера

Для запуска проекта необходимо использовать локальный сервер с поддержкой PHP и MySQL.

Подойдут следующие решения:

- OpenServer
- XAMPP
- Denwer
- встроенный сервер PHP

Папку проекта необходимо разместить в директории веб-сервера (например `htdocs` или `domains`).

### 3. Создание базы данных

Откройте **phpMyAdmin** и создайте новую базу данных.

Например:

blog_db

### 4. Импорт базы данных

В папке проекта находится файл базы данных:

database/blog.sql

Необходимо открыть phpMyAdmin и выполнить импорт этого файла в созданную базу данных.

### 5. Настройка подключения к базе данных

Откройте файл:

config/db.php

И укажите параметры подключения к вашей базе данных:

```php
$host = "localhost";
$dbname = "blog_db";
$username = "root";
$password = "";

Параметры могут отличаться в зависимости от используемого сервера.

6. Запуск проекта

После выполнения всех настроек сайт можно открыть в браузере:

http://localhost/название_папки_проекта 

Если используется OpenServer, адрес может быть:

http://blog/
